# Proposta Tecnica

## Problema
Apoiar monitoramento climatico com dados espaciais para gerar impacto positivo em operacoes na Terra.

## Objetivo do MVP
Construir uma POC com ingestao de dados, previsao por IA e visualizacao em dashboard.

## Escopo inicial
- Ingestao de dados publicos
- Baseline de ML
- Dashboard com KPIs e previsao

## Escopo incremental
- Modulo de visao computacional
- Integracao ESP32
